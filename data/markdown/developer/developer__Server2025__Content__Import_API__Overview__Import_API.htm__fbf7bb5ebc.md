# Import API

The Relativity Server Import SDK .NET package provides functionality for programmatically importing large numbers of documents, images, and Relativity Dynamic Objects (RDOs) into a Relativity workspace.

See these related pages:

- Get started with the Import API

- Import API basic concepts

- Import API best practices

- Handle errors and exceptions

- Troubleshoot the Import API

Due to deprecation, all Relativity.DataExchange.Client.SDK package references must be migrated to Relativity.Server.Import.SDK
