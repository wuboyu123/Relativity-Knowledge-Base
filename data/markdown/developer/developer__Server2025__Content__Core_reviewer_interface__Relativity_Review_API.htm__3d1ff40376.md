# Review API

The Review API is a suite of APIs that you can use to build and maintain Review integrations. Review is an updated Relativity document review application that you can use to review, annotate, and code documents in a workspace. It offers performance and extensibility improvements over the current HTML viewer.

The API supports the extensibility of Review by providing JavaScript classes and functions with the following features:

- Highlight and Redaction APIs - support integrations used for creating and editing redactions and highlights. They also listen for user-created redactions and highlights.

- Zoom APIs - are used for setting the zoom level and zoom mode.

The Review API and Review Extension framework are compatible with RelativityOne and Relativity Server 2022 and later.

For more information, see the Review API reference documentation on the Review API (Relativity Server 2025) site .
