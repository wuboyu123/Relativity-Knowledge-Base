# Agent Manager

The Agent Manager API exposes multiple operations that you can use to programmatically manage agents in your Relativity environment. In Relativity, agents are process managers and workers that run in the background to complete jobs scheduled in your environment. For general information, see Agents on the Relativity Documentation site.

Common use cases:

- Adding new worker agents when multiple jobs are submitted to the OCR or Branding queues.

- Deleting worker agents that are unresponsive.

- Developing a custom monitoring tool for messages logged by agents.
