# Infrastructure

Infrastructure APIs enable you to manage operations within underlying Relativity servers and infrastructure.

## Current Infrastructure API documentation

#### Agent Manager API documentation

The Agent Manager API exposes multiple operations that you can use to programmatically manage agents in your Relativity environment.

- Agent Manager (.NET) / Agent Manager (REST)

#### Notification Manager API documentation

The Notifications Manager API exposes methods for reading, updating, and dismissing the MotD. It also includes methods for determining whether the MotD has been dismissed, and whether it is displayed as plain text or HTML.

- Notifications Manager (MotD) - (.NET) / Notifications Manager (MotD) - (REST)

#### Resource server / resource pool API documentation

The Resource APIs provide API endpoints to interact with Relativity servers. The type of interaction is based on the type of server. All server types support read and update operations. Some server types also support create and delete operations, console actions, and other operations.

- Resource Server / Resource Pool (.NET / REST)
