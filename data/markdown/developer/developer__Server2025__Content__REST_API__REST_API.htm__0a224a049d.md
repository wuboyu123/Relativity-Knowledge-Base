# REST API

With the introduction of the REST API, Relativity further extends the development options available through the platform. The Relativity REST API is designed to streamline the development of mobile, web-based, and cross-platform applications by leveraging standard features of the Representational State Transfer (REST) architectural model. It provides you with the ability to interact with Relativity resources using the HTTP protocol for transferring requests and responses between client and server.
