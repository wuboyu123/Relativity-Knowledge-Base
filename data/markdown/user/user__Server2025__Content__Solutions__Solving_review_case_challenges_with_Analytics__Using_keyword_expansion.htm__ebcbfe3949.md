# Using keyword expansion

The steps required to identify relevant terms currently unknown in your document set involve using the Analytics Keyword Expansion feature. In this scenario, you already identified a starting list of keywords in your data set.

Use either of the two following methods to initiate a keyword expansion search to discover additional key terms applicable to your case review:

- Perform a keyword search from the Conceptual Keyword Expansion dialog.

- Perform a keyword search with selected text in the document Viewer.

Keyword expansion requires an Analytics index. See Creating an Analytics index.

## From the Documents tab

Follow these steps to use keyword expansion from the Documents tab:

- Select the Documents tab.

- Select an Analytics index from the search index drop-down menu.

- Click Expand next to the Concepts search box.

- Type a keyword in the search box with which you wish to perform a keyword expansion search.

- Click Expand .

The Conceptual Keyword Expansion dialog lets you easily review the related keywords identified by Analytics. You also have the option to click the hyperlink for each related keyword in order to perform a keyword expansion search for a related term.

## From the Viewer

Follow these steps to use keyword expansion from the Viewer:

- Open a document from the Documents tab.

- In the Viewer, select text in the document with which you wish to perform a keyword expansion search.

- Right-click on the selected text and select Keyword Expansion

The Conceptual Keyword Expansion dialog lets you easily review the related keywords identified by Analytics. You also have the option to click the hyperlink for each related keyword in order to perform a keyword expansion search for a related term.
