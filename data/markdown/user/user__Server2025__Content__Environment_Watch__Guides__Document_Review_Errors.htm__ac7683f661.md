# Document Review Errors

This dashboard provides visibility into document loading success and failure rates within Relativity's Review Interface. It helps quickly identify patterns in viewer errors, determine whether failures are isolated or widespread, and ensure that documents load and display reliably during review. These insights help troubleshoot failed document loads, verify service uptime, and confirm that the document review experience remains stable across all workspaces.

## Error Distribution and Message Insights

The Error Distribution and Message Insights section visualizes the types and frequency of viewer errors encountered in Relativity. The Error Messages chart highlights the most common document-related failures, including unsupported formats or unrecognized file types. The Coding Pane Errors by Message chart provides visibility into errors that occur during coding operations within the review interface. Together, these visualizations help identify whether specific document types, file formats, or actions are contributing to viewer instability or failed document loads. Monitoring these patterns enables timely troubleshooting and performance optimization. The Error Rate graph tracks the percentage of document loads that resulted in errors over time. Spikes in error rate can indicate temporary viewer instability, conversion issues, or misconfigured services. The 'Document Not Loading and Skeleton Shown' metric captures instances where the document failed to load fully, showing only a skeleton or placeholder instead of the expected content. Monitoring this metric helps confirm that all documents are rendering as intended.

### Use Cases

Use Case Description

Identify repeated viewer errors Detect recurring error types or unsupported file formats that cause document load failures, enabling faster root cause analysis and remediation.

Troubleshoot rendering issues Pinpoint specific error messages and affected file types when reviewers report documents not loading correctly or appearing incomplete.

Ensure smooth review performance Regularly review error frequency and message distribution to maintain a consistent and reliable document viewing experience across all Relativity workspaces.
