# Antivirus

This dashboard provides visibility into antivirus status information reported by each monitored host. It shows whether a supported antivirus product is detected and surfaces key status indicators to help administrators understand the current security posture of their Relativity infrastructure.

This dashboard provides visibility into antivirus status as reported by the host. It is intended for monitoring and investigation purposes. Currently, it supports only Windows Defender Antivirus.

## Antivirus Status By Host

This dashboard provides visibility into antivirus status information reported by each monitored host. It shows whether a supported antivirus product is detected and surfaces key status indicators to help administrators understand the current security posture of their Relativity infrastructure.

### Use Cases

Use Case Description

Verify antivirus presence Review which hosts report an installed and running antivirus product.

Monitor reported protection state Identify hosts where real-time protection is reported as disabled or not running.

Identify potential update issues Spot hosts that report antivirus definitions as out of date for further investigation.

Investigate reported security gaps Use reported status indicators to guide follow-up actions with infrastructure or security teams.
