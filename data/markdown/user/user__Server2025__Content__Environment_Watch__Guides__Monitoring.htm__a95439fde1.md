# Monitoring

The Monitoring domain provides observability into the health and performance of Relativity environments by collecting and analyzing telemetry data from all servers. It focuses on metrics, logs, and traces to help administrators proactively detect issues, troubleshoot problems, and maintain system stability. Monitoring agents are responsible for gathering metrics and telemetry across distributed components, ensuring comprehensive visibility into system behavior.
