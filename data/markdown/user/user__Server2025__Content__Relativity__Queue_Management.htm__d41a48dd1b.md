# Queue Management

The Queue Management tab is the parent tab of the following queue sub-tabs:

## Worker manager queue

The Worker Manager Server handles all imaging, conversion, and processing jobs. The Worker Manager Queue is where you can monitor all processing jobs and imaging jobs.

Conversion jobs can't be monitored from any queue.

For more information on managing imaging jobs, see Imaging jobs in the Worker Manager Queue .

For more information on managing processing jobs, see Managing processing jobs in the queue .

## Branding queue

The Branding queue is where you can monitor all Branding jobs across an instance of Relativity. See Branding queue for more information.

## dtSearch queue admin

Monitor dtSearch indexing jobs across an instance of Relativity. See dtSearch queue admin for more information.

## OCR queue

The OCR queue is where you can monitor all OCR jobs across an instance of Relativity. See OCR queue for more information.

## PDF queue

Monitor PDF jobs running across an instance of Relativity. You can cancel a job using the mass operation menu options. See PDF Queue for more information.

## Workspace upgrade queue

The Workspace Upgrade queue is where you can monitor many aspects of an upgrade across an instance of Relativity. See Upgrading workspaces for more information.

## Production queue

The Production queue is where you can monitor all Production jobs across an instance of Relativity. See Production queue for more information.
