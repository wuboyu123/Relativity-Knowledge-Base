# Data Transfer

The Data Transfer domain provides observability into workflows that move data into, out of, and within a Relativity Server environment. It focuses on monitoring the health, performance, and reliability of integration processes that handle document, image, and file transfers. This domain helps administrators ensure smooth data exchange and troubleshoot issues that could impact system operations.

## Integration Points

Integration Points manage the import and export of data across connected systems. Monitoring these processes is critical for maintaining consistent performance and preventing disruptions in data movement.

- Integration Points Overview

- Integration Points Job Errors

- Mass Import Overview
