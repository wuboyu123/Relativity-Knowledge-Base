# Eventing

The Eventing domain provides observability into Relativity's messaging infrastructure, specifically RabbitMQ. This domain focuses on monitoring RabbitMQ health, performance, and queue behavior to ensure reliable communication across distributed services.

## RabbitMQ

Relativity-built dashboards offer visibility into RabbitMQ cluster health, message flow, and queue activity. These dashboards help monitor throughput, identify bottlenecks, and maintain system responsiveness.

Available dashboards:

- RabbitMQ Overview

- RabbitMQ Queue Detail
