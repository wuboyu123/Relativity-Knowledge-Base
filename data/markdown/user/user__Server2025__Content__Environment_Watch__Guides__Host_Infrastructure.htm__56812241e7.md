# Host Infrastructure

The Host Infrastructure domain provides observability into the core systems supporting Relativity, focusing on the performance, stability, and health of host-level resources across the environment.

Relativity-built dashboards offer detailed insights into host infrastructure, including storage, processes, network, fileshares, certificates, Windows services, and monitoring agents. These dashboards provide a comprehensive view of host performance and operational health across multiple hosts.

## Available Dashboards

- Host Infrastructure Overview

- Storage

- Processes

- Network

- Relativity Fileshares

- Certificates

- Windows Services

- Antivirus
