# Mailbox

The Mailbox tab provides you with a comprehensive list of all emails, incoming and outgoing, and corresponding details in every project in the workspace.

The Mailbox also shows incoming messages so that you can track out of office notifications, bounce backs, custodian responses, etc. Use the column filters to sort information and click on an item to view its details.

## Accessing the mailbox

Access the Legal Hold Mailbox via the Mailbox tab or within a project on the Project Details page. For more information, see Projects .
