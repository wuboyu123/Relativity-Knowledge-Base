# Elastic Stack manual installation

Manual installation installs Elastic Stack components individually and allows you to configure each component directly.

This approach is intended for advanced or specialized scenarios where automated installation does not meet specific requirements, such as environments that require custom installation workflows or non-standard deployment architectures.

If you are installing Elastic Stack for the first time and do not have specific customization requirements, start with automated installation.

## Components

Manual installation is organized by component:

### Elasticsearch

Required component for both Data Grid Audit and Environment Watch.

### Kibana

Required for Environment Watch. Optional for Data Grid Audit.

### APM Server

Required for Environment Watch. Not used for Data Grid Audit.
