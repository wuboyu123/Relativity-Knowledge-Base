# Mass move

You can use the mass move operation to move selected documents from their current folder to another existing folder. If you mass move any documents, any custom security applied to those documents is lost. The document inherits the rights of the new parent folder. Mass move is available from the mass operations bar.

You can't use mass operations on Data Grid-enabled fields.

- From the mass operations bar on the document list, choose whether to move All searchable items, Checked items or These items in the current returned set.

- Select Move in the drop-down menu.

The mass move select folder form displays.

- Select the target location and click OK . Note that all documents move to the target directory. The subdirectory structure of the source folder is maintained.
