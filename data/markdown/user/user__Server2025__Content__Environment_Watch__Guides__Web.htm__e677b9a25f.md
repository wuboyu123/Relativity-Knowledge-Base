# Web

The Web domain provides observability into Relativity's web-facing components, focusing on the health, performance, and responsiveness of IIS-hosted services and other web-layer processes that power the Relativity user experience. Dashboards in this domain help administrators ensure that web servers, application pools, and related endpoints are operating efficiently, responding quickly, and handling requests without error.

## IIS

Relativity-built dashboards under the IIS category provide detailed insights into the stability, performance, and error conditions of IIS web servers across the Relativity environment. These dashboards enable proactive monitoring of request activity, response times, and system resource utilization to maintain optimal web-tier health.

Available dashboards:

- IIS Overview

- IIS Instance Overview

- IIS Errors

- IIS Latency
