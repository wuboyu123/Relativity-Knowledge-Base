# Analytics

The Analytics feature domain gives visibility into how Relativity's advanced search and analysis tools are performing. It currently focuses on dtSearch indexes, providing insight into full-text search performance, index size, and operational health. These components help relativity administrators monitor query efficiency, index integrity, and overall search reliability within a workspace. While the Analytics feature domain includes capabilities such as conceptual indexing, clustering, categorization, concept search, keyword expansion, and document similarity, Environment Watch currently provides monitoring visibility for dtSearch indexes, with additional analytics monitoring planned for future releases.

## dtSearch

Monitors the status and performance of dtSearch indexes, ensuring efficient query execution and index integrity across environments.

Available dashboards:

- dtSearch Index Overview

- Search Terms Report Overview
