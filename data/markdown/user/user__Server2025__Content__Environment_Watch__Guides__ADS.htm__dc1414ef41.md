# ADS

The ADS domain provides observability into key Relativity services and components that support application functionality and distributed operations. It focuses on monitoring agent activity, service host health, and Kepler service performance to ensure stability and efficiency across the environment.

Available dashboards:

- ADS Overview

- Agents Overview

- Service Host Overview

- Kepler Services Overview

- Kepler Service Requests
