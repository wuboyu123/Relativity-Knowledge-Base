# Service Host Overview

This dashboard provides a centralized view of the operational health and resource consumption of service hosts within a Relativity environment. It tracks CPU, memory, and disk activity across hosts, helping relivity administrators validate deployments, monitor performance, and identify resource constraints that could impact system stability. This dashboard is essential for routine health checks, post-installation validation, and diagnosing performance issues related to service host operations.

## Service Host Resource Summary

This section provides a consolidated health indicator for all service hosts, along with key metrics such as CPU utilization, memory usage, and disk operations. The accompanying table enables relativity administrators to compare resource consumption across hosts, making it easier to detect anomalies or hosts under stress. Trends in CPU and memory utilization help pinpoint potential bottlenecks, guide capacity planning, and ensure that service hosts maintain optimal performance under varying workloads.

## Memory and Disk Utilization Trends

This section visualizes memory utilization and disk operations over time for each service host. Monitoring these trends helps relativity administrators identify patterns such as sustained high memory usage or spikes in disk activity that may signal resource contention or inefficient processes. By correlating these metrics with workload changes or system events, relativity administrators can proactively address issues before they affect overall performance.

## Use Cases

Use Case Description

Validate service host health Confirm that all service hosts are operating within acceptable resource thresholds after installation or configuration changes.

Monitor resource utilization Track CPU, memory, and disk activity to ensure hosts are not overburdened and maintain optimal performance.

Detect anomalies Identify unusual spikes or sustained resource usage that may indicate underlying issues requiring investigation.

Troubleshoot performance issues Use historical trends and comparative metrics to isolate problematic hosts and guide remediation efforts.
