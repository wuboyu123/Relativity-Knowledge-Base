# Review

The Review domain provides observability into the Relativity Review Interface, focusing on document viewing, conversion performance, and reviewer experience. Dashboards in this domain help monitor the efficiency and stability of the document review workflow, ensuring that documents render quickly, convert reliably, and display without error during review.

## Document Review

Dashboards within Document Review provide visibility into document viewer performance, caching efficiency, error rates, and reviewer activity. They enable proactive monitoring of the review experience, helping identify latency issues, viewer failures, and conversion bottlenecks that may affect document rendering.

Available dashboards:

- Document Review Overview

- Document Review Performance

- Document Review Conversion

- Document Review Errors

## Conversion

Dashboards within Conversion provide insight into document conversion activity, cache utilization, and processing performance across the Relativity environment. They support performance tuning and verification of the conversion processes that prepare documents for viewing in the Review Interface.

Available dashboards:

- Conversion Overview
