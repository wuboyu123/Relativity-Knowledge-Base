# Integration Points Job Errors

This dashboard privides a consolidated, operational view of failed Import and Export jobs executed through Relativity Integration Points. It helps administrators quickly identify integration failures, understand where issues are occurring, and take timely corrective action.

## Failed Import/Export Jobs Overview

This section highlights Import/Export jobs that did not complete successfully. By associating Integration Points IDs with specific Job IDs, it allows administrators to quickly identify which job encountered errors. This information supports rapid investigation of issues related to data formatting, permissions, connectivity, or configuration problems.

### Use Cases

Use Case Description

Troubleshoot failed jobs Quickly identify failed import or export jobs and take corrective action to restore data transfer workflows.

Verify post-upgrade stability Validate that Integration Points continue to function correctly after Relativity upgrades or configuration changes.
