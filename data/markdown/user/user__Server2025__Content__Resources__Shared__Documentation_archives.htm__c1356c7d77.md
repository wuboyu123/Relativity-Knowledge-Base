# Documentation archives

The content on this website is for Relativity 2025 . If you're using an earlier version of Relativity, use the documentation from the appropriate version. If you're using an earlier, unsupported version, click the Help link from the user drop-down menu in your instance of Relativity.

If you need further technical assistance, contact our Support team through our Support page .
