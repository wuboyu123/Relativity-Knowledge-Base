# Feature Domain

## Context

This document defines the official feature domains defined for Relativity Server used by the Environment Watch product to define standard observability metrics/traces/logs, combined with alerts and health check results, that are used to group various 'signals' within top-level visualizations to provide the end user a simple way to assess the overall instance health.

## Official Domains

Domain # Name Examples

01 ADS Workspace upgrade, Application installation

02 Agents Integration Points Agent, Conversion Cache Discovery Agent

03 Analytics Structured/Conceptual/CAAT

05 Billing and Licensing Relativity/Invariant license expired
