# Case permission audit report

For a given Relativity instance, the Case Permission Audit Report script generates a report that lists all users, workspaces, and groups, as well as each group's permissions. This report helps system admins manage their environment's permissions.

## Special considerations

Consider the following when working with the Case Permission Audit Report script.

This script:

- Returns results first ordered by username (<first name> <last name>) and then by group name.

- Does not show the System Administrators or Personal groups.

- Supports distributed SQL Servers.

- Only runs from Home.

We recommend running this during off hours. Do this during off hours especially if the environment is large as this queries the entire instance, all users, and all workspaces.

## Inputs

None. This script runs across all workspaces.

## Outputs

This script generates the following outputs:

- Group names

- Group permissions

- Group users

- Workspaces

## Running the script

To run the script:

- Click your name in the upper right corner of Relativity, and click Home .

- Click the Relativity Script Library tab.

- Click the Case Permission Audit Report link.

- On the script details page, click the Run Script button.

- In the Run Script window, click Run again.

For additional assistance, contact Relativity Customer Support .
