# Imaging history

You can view the audit history when a job is run for an imaging set, mass image, or image-on-the-fly job by going to the History tab and filtering for Imaging in the name column.

If you have the Data Grid for Audit application installed, the History tab is hidden.

Use the following guidelines to view imaging history records:

- Imaging Set Jobs - the Name column displays the name of the imaging set, and the Object Type column displays ImagingSet.

- For each image created from the set, the Name column displays the document name, and the Object Type column displays Document. Click Details to display a pop-up containing the name of the imaging profile and the formatting options used to image a document. (The formatting options displayed on the pop-up list those used when the imaging job was run. They don't reflect any subsequent changes to the imaging profile.)

- Image-on-the-Fly or Mass Image Jobs - the Name column displays the name of the imaging profile, and the Object Type column displays ImagingProfile.

For each image created, the Name column displays the document name, and the Object Type column displays Document. Click Details to display a pop-up containing the name of the imaging profile and the formatting options used to image a document as described above.
