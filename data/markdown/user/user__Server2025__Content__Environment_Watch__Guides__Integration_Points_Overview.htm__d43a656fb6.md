# Integration Points Overview

This dashboard privides a centralized view of data moving in and out of Relativity and across Relativity workspaces using Integration Points. It monitors the status of Integration Points, tracks job success rates, and highlights any issues with document, image, or file transfers. This dashboard is essential for ensuring smooth data movement and maintaining system reliability.

## Integration Points Health and Job Metrics

The dashboard provides a comprehensive summary of integration points health and job activity. It displays the overall status of Integration Points, success rates for import and export jobs, and counts of documents and images processed. These insights help users assess reliability, detect bottlenecks, and maintain efficient data exchange processes.

### Use Cases

Use Case Description

Troubleshoot failed jobs Quickly identify failed or stuck jobs and take corrective action to restore data transfer workflows.

Monitor integration health Validate that Integration Points are healthy and performing as expected, reducing risk of data exchange disruptions.

Analyze integration performance Review success rates and job counts to assess integration reliability and optimize system performance.

- Integration Points Job Failure

- One or More Integration Points Jobs Are Stuck
