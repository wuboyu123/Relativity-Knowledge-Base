# Host Infrastructure Overview

This dashboard provides a comprehensive view of host-level metrics, including CPU usage, memory utilization, disk I/O, and network performance. It helps administrators monitor resource consumption, detect anomalies, and maintain the overall health and performance of hosts.

## Overall Resource Utilization

The dashboard displays a table summarizing average resource usage across hosts, including Host, Server Type, Host CPU Utilization, Host RAM Utilization, and Host Disk Utilization. Additionally, a graph shows top CPU utilization per host over time, with the horizontal axis representing timestamps, allowing users to identify high-load hosts quickly.

## Memory and File System Utilization

The dashboard provides detailed insights into memory usage and file system utilization across hosts. Two graphs show top memory utilization per host and file system utilization per host over time, helping administrators detect resource bottlenecks or capacity issues.

### Use Cases

Use Case Description

Monitor host resource utilization Monitor host resource utilization to proactively identify CPU, memory, or disk bottlenecks.

Identify top resource-consuming hosts Track top resource-consuming hosts over time to optimize performance and load balancing.

Analyze capacity and usage trends Investigate trends in memory and file system usage to prevent capacity issues.

Respond to resource threshold alerts Respond quickly to alerts related to resource thresholds to maintain system stability.

### Associated Alerts

- CPU has been exceeding threshold on at least one host

- Disk IO has been exceeding threshold on at least one host

- Memory is exceeding threshold on at least one host
