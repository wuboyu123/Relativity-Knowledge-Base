# Search Terms Report Overview

This dashboard provides operational insight into Search Terms Report (STR) processing within your Relativity environment. Search Terms Report jobs validate search strategies by calculating term hits across document sets. The dashboard enables administrators to quickly assess STR job health, identify failures, detect queue bottlenecks, and monitor response performance over time.

## STR Job Health and Performance Overview

The dashboard provides a consolidated view of STR job health across the selected time range. Key metrics include:

- Job Success Rate - Validates overall STR job reliability.

- Completed Jobs (Total) - Indicates STR processing volume.

- Failed Jobs (Total) - Highlights processing errors requiring investigation.

- Median Job Duration - Reflects typical execution time.

- Long Queued Jobs - Identifies potential agent or resource bottlenecks.

- Median Term Response Time - Measures how quickly search terms return results.
