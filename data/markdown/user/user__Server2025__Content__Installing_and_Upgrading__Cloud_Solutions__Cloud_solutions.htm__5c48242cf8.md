# Cloud solutions

Many organizations are now choosing to use Relativity in the cloud to minimize the cost and effort associated with managing server infrastructure Relativity Server .

As always, you can rely on Relativity’s partner network of service providers to host your data while providing diverse, expert services—from project or administrative management, to end user support and training, to throughput analysis and quality assurance.

You can also bring the power of Relativity in house using RelativityOne . RelativityOne delivers all the features and functionality that set Relativity Server apart, wrapped in an easy-to-manage cloud solution that you can use with or without an existing Relativity Server solution.
